# assignment-09 (SkillSwap)

Live URL: [(click to view)](https://assignment-09-taupe.vercel.app/)

## Purpose

This is a skill-sharing marketplace demo built with React + Vite. Users can browse skills, view skill details, sign up / log in (Firebase Auth), and book sessions. The project demonstrates modern front-end tooling (Tailwind CSS, DaisyUI, Framer Motion) and Firebase authentication.

## Key features

- Browse skill listings with cards
- Dynamic skill details pages
- Authentication (Email/password + Google Sign-In) using Firebase
- Protected routes (private routes)
- Responsive layout using Tailwind CSS + DaisyUI
- Scroll animations (AOS + Framer Motion)
- Toast notifications with react-toastify
